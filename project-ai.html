AI Ассистент

Умный чат-бот на основе LLM с долгосрочной памятью, инструментами и семантическим поиском по документам.

Смотреть на GitHub → Демо ↗

О проекте

Это продакшен-готовый AI-ассистент, который умеет работать с документами, запоминать контекст разговора и вызывать внешние инструменты. Не просто обёртка над ChatGPT — полноценная агентная архитектура.

Ассистент использует RAG (Retrieval-Augmented Generation): перед ответом он ищет релевантные фрагменты в векторной базе данных и добавляет их в контекст. Это позволяет работать с большими корпусами документов без потери точности.

Ключевые возможности

🧠
Долгосрочная память
Помнит предыдущие разговоры, персонализирует ответы на основе истории взаимодействия.
🔍
RAG поиск
Семантический поиск по базе документов через векторные эмбеддинги. Точные ответы из вашего контента.
🛠️
Tool calling
Вызов внешних API, поиск в интернете, выполнение кода — агент сам решает, какой инструмент использовать.
⚡
Стриминг ответов
Ответы появляются токен за токеном — никаких долгих ожиданий, ощущение живого диалога.

Пример кода

# RAG pipeline — поиск + генерация ответа async def answer_with_rag(query: str, session_id: str): # 1. Семантический поиск по документам docs = await vector_store.similarity_search( query, k=5, filter={"session": session_id} ) # 2. Загрузка памяти из Redis history = await memory.load_messages(session_id) # 3. Генерация ответа с контекстом async for chunk in llm.astream({ "context": docs, "history": history, "question": query, }): yield chunk.content
